# printerpy

**PRINTERPY**

**WHAT IS THAT?**

a simple python program to type text like from your keyboard
with a simple visualisation of your text

**FOR WHAT?**
if you use virtual web console / problems with connection / putty errors / etc

**HOW IT WORKS?**

put your text in "input.txt" file
the program will do all the "\n" n stuff by itself

find a input where you need to type a text and type y (Y) in program
program will start typing!

**ENJOY!**
